# brain3-mcp

MCP (Model Context Protocol) server for BRAIN 3.0 — an AI-powered personal operating system designed for ADHD.

This server exposes BRAIN 3.0's core data (goals, projects, tasks, routines, check-ins, and activity patterns) as MCP tools and resources, enabling Claude to interact with the system as an intelligent partner.

## Status

Under construction. Phase 1 implementation in progress.
